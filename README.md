# AccountLab
AccountLab is a Java application built to practice unit testing and clean domain modeling. 
The project demonstrates object-oriented design, immutable transaction records, business rule validation, and separation of responsibilities within a simple banking context.

## Project Goals
- Practice unit testing with JUnit
- Model a clean banking domain
- Implement immutable transaction history
- Apply separation of responsibilities (Account + Transaction + Bank)
- Handle business rules and validation
- Design testable code

## Technologies Used
- Java 25
- Maven
- JUnit 5
- Git & GitHub

## Features
- Create bank accounts
- Deposit money
- Withdraw money
- Validate insufficient balance
- Immutable transaction history
- Timestamped transactions

## How to Run

1. Clone the repository
2. Run: mvn test

## Concepts Practiced

- Object-Oriented Programming
- Domain modeling
- Immutability
- Exception handling
- Service layer design
- Testable architecture design
